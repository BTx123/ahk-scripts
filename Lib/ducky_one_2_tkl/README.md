# AutoHotKey Scripts for Duck One 2 TKL

## Mouse
| Action      | Shortcut   |
|:------------|------------|
| Right Click | `Alt`+`F1` |

## Music
| Action     | Shortcut   |
|:-----------|------------|
| Play/Pause | `Alt`+`F1` |
| Previous   | `Alt`+`F2` |
| Next       | `Alt`+`F3` |

## Volume
| Action      | Shortcut    |
|:------------|:------------|
| Volume Up   | Pause       |
| Volume Down | Scroll Lock |
