# ahk-scripts
AutoHotKey Scripts

## Music Controls
| Action     | Shortcut   |
|------------|------------|
| Play/Pause | `Alt`+`F1` |
| Previous   | `Alt`+`F2` |
| Next       | `Alt`+`F3` |

## Volume Controls
| Action      | Shortcut    |
|-------------|-------------|
| Volume Up   | Pause       |
| Volume Down | Scroll Lock |
