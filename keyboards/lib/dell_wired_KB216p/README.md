# AutoHotKey Scripts for Dell Wired KB216p

## Music

| Action     | Shortcut   | Description                         |
| :--------- | :--------- | :---------------------------------- |
| Play/Pause | `Alt`+`F1` | Toggle audio between play and pause |
| Previous   | `Alt`+`F2` | Go to previous song                 |
| Next       | `Alt`+`F3` | Go to next song                     |
