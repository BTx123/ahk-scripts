# AutoHotKey Scripts for DasKeyboard 4 Pro

## Discord

| Action | Shortcut           | Description                |
| :----- | :----------------- | :------------------------- |
| Mute   | `Ctrl+Alt+Shift+M` | Mute yourself on Discord   |
| Deafen | `Ctrl+Alt+Shift+D` | Deafen yourself on Discord |
